CSS Gradients

The gradient CSS data type is a special type of image that consists of a transition between two or more colors

gradient types

  • linear-gradient(#7fffd4, #683af9);
  • radial-gradient(#7fffd4, #683af9);
  • repeating-linear-gradient(#7fffd4, #683af9 50px);
  • repeating-radial-gradient(#7fffd4, #683af9 50px);
  • conic-gradient(#7fffd4, #683af9);

direction

The linear gradients and conical can include a direction. For example:

  • linear-gradient(to bottom right, tomato, powderblue);
  • linear-gradient(45deg, tomato, powderblue);
  • linear-gradient(0.25turn, tomato, powderblue);
  • conic-gradient(from 45deg, #7fffd4, #683af9);

stop position

You can add stop positions to shift the mid-point of the transition between two colors as well as set when colors start and stop. For example:

  • linear-gradient(to left, tomato, 20%, powderblue);
  • linear-gradient(red 70%, orange 80%, yellow 90%);
  • linear-gradient(red 0 33%, orange 33% 66%, yellow 66% 100%);

multiple gradients

You can layer up gradients that have transparency in their rgba values. for example:

  • background:
    linear-gradient(to top, rgba(0,0,0,1), rgba(0,0,0,0) 95%),
    linear-gradient(to left, rgba(255,0,0,1), rgba(255,0,0,0));
  • background:
    linear-gradient(217deg, rgba(255,0,0,.85), rgba(255,0,0,0) 75%),
    linear-gradient(127deg, rgba(0,255,0,.85), rgba(0,255,0,0) 75%),
    linear-gradient(336deg, rgba(0,0,255,.85), rgba(0,0,255,0) 75%);

center point

for conical gradients, you can set the center point by using the at keyword. For example:

  • conic-gradient(at 20% 100%, honeydew, powderblue);
  • repeating-conic-gradient(at 20% 100%, honeydew 0 9deg, powderblue 9deg 18deg);
