Explicit Grid.

CSS Grid Layout is the most powerful layout system available in CSS. It is a 2-dimensional system, meaning it can handle both columns and rows, unlike flexbox which is largely a 1-dimensional system. You work with Grid Layout by applying CSS rules both to a parent element (which becomes the Grid Container) and to that elements children (which become Grid Items).

Grid Container Properties.

  • dispay: grid;
  • grid-template-columns: 350px 1fr 150px;
  • grid-template-rows: 50px auto 100px;
  • grid-template-areas: 'left middle right'
  • grid-auto-rows: 100px;
  • grid-auto-columns: 100px;
  • grid-auto-flow: column; (or 'row', 'row dense', 'column dense)
  • grid-column-gap: 5px;
  • grid-row-gap: 5px;
  • grid-gap: 5px;
  • /* (Shorthand) grid-template: grid-template-rows / grid-template-columns */
  • grid-template: 50px auto 100px / 350px 1fr 150px;
  • /* (Shorthand) grid-template: 'grid-template-areas' template-row 'grid-template-areas' template-row / grid-template-columns */
  • grid-template: 'side nav-l nav-r' 50px 'side main-l main-r' auto 'foot foot foot' 100px / 1fr 2fr 2fr;

Grid Item Properties.

  • grid-column-start: 3;
  • grid-column-end: span 3;
  • grid-row-start: 2;
  • grid-row-end: 4;
  • /* (Shorthand) grid-column: column-start / column-end */
  • grid-column: 3 / span 3;
  • /* (Shorthand) grid-row: row-start / row-end */
  • grid-row: 2 / 4;
  • /* (Shorthand) grid-area: row-start / column-start / row-end / column-end */
  • grid-area: 2 / 4 / 3 / span 3;

Named areas.

CSS Grid Layout excels at dividing a page into major regions or defining the relationship in terms of size, position, and layer, between parts of a control built from HTML primitives. Like tables, grid layout enables an author to align elements into columns and rows. However, many more layouts are either possible or easier with CSS grid than they were with tables. For example, a grid container's child elements could position themselves so they actually overlap and layer, similar to CSS positioned elements.
