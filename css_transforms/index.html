CSS Transform

The transform property lets you rotate, scale, skew, or translate (reposition) an element. It modifies the coordinate space of the CSS visual formatting model.

There are many transform functions that can be used. The most basic ones are as follows...

rotate()

Rotates an element around a fixed point on the 2D plane.

rotate(10deg)

scale()

Scales an element up or down on the 2D plane.

scale(0.75, 0.75)

skew(25deg, 5deg)

Skews an element on the 2D plane.

scale(25deg, 5deg)

translate()

Repositions an element on the 2D plane.

translate(50%, 25px)

perspective()

Sets the distance between the user and the z=0 plane. In order to see perspective, you need to have also applied another transformation like rotate.

rotate(-15deg) rotateY(30deg)
rotate(-15deg) rotateY(30deg) perspective(250px)

matrix()

Describes a homogeneous 2D transformation matrix. The values represent the following functions:

matrix( scaleX(), skewY(), skewX(), scaleY(), translateX(), translateY() )

matrix(1.4, 0.25, 0.25, 1.4, 100, 50)
